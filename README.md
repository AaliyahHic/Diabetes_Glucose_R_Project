# Diabetes Glucose Analysis in R

## Overview
This project analyzes simulated diabetes data using R. It focuses on blood glucose readings, carb intake, and insulin units to explore patterns over time and by time of day.

## Files
- diabetes_glucose_analysis.R
- diabetes_data.csv
- summary_stats_diabetes.csv (generated after running script)

## Skills Demonstrated
- Data cleaning and transformation
- Visualization with ggplot2
- Linear regression
- Summary statistics
